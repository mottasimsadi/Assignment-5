Question-1: What are the different ways to select an element in the DOM?

There are several ways to select elements in the DOM using JavaScript:

1. By ID: It selects a single element by its unique id attribute.
Example: document.getElementById('elementId');

2. By Class Name: It selects all the elements with a specific class name.
Example: document.getElementsByClassName('className');

3. By Tag Name: It selects all the elements with a specific tag name.
Example: document.getElementsByTagName('div');

4.By Query Selector: It selects the first element that matches a query selector.
Example-1: document.querySelector('.className');
Example-2: document.querySelector('#idName');

5. By Query Selector (All): It selects all the elements that match a query selector.
Example: document.querySelectorAll('.className');

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText & textContent:

innerHTML :
~ It basically gets or sets the HTML content inside an element.
~ It includes all the HTML tags and text.

innerText :
~ It basically gets or sets the visible text content of an element, excluding any HTML tags.
~ It can't show or return the hidden text.

textContent :
~ It basically gets or sets the text content of an element, including all child nodes, but excludes HTML tags.
~ It can show or return the hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique for handling events by assigning them to a parent element instead of attaching event listeners directly to child elements. If you have many child elements that should process the same event or if child elements will be added or removed, that may be helpful.

Question-4: What is event bubbling in the DOM?

The event bubbling is a dom mechanism that sends the event from the target element (element that triggered the event) up the dom tree on all its ancestors. This means that the event will be fired first on the child followed by its parent till the root of the document if you click on a child element.

Question-5: How do you create, add, and remove elements using JavaScript?

How to create an element, add an element and delete an element using JavaScript is answered below:

1. Create a new element: Using createElement(), a new element can be created.
Example: document.createElement("tagName")

2. Add a new element: Using appendChild(), the element can be added that was created.
Example:
let parent = document.getElementById("container");
parent.appendChild(new element name);

3. Deleting an element: Using removeChild(), any element can be deleted.
Example:
let parent = document.getElementById("container");
parent.removeChild(element name);
